Bedienungsanleitung

Für den Spielleiter

Erstelle eine Kampfszene über "+ Neu". Füge anschließend NSCs aus der Bibliothek hinzu oder importiere Spielercharaktere über ihre Codes. Für jeden Charakter kannst du:

  • Schaden austeilen (klicke auf die Zustandsmonitor-Kreise)
  • Panzerung verwalten (blaue Kreise)
  • Würfelmali werden automatisch berechnet
  • Edge, Plotpunkte und Karma tracken
  • Werte jederzeit anpassen

Für Spieler

Erstelle deinen Charakter im Spieler-Tab. Er wird im Browser gespeichert und bekommt automatisch einen 8-Zeichen-Code (z.B. NOVA3R7X).

Du kannst deinen Charakter auf drei Arten teilen:

  • 8-Zeichen-Code - kurz und tippbar. Braucht Cloud-Anbindung (siehe ⚙ Einstellungen).
  • QR-Code - auf dem anderen Gerät einscannen. Funktioniert offline zwischen Geräten in der Nähe.
  • Langer Code - Base64-String zum Kopieren. Funktioniert ohne Cloud, z.B. per Chat.

Cloud-Einrichtung (optional, für 8-Zeichen-Codes)

Damit die kurzen Codes zwischen Geräten funktionieren, brauchst du einen kostenlosen Account bei JSONBin.io. Klicke oben auf ⚙ und folge den Schritten dort. Die Einrichtung ist einmalig, dauert 2 Minuten, und alle Spieler können denselben Key verwenden.

Ohne Cloud funktionieren weiterhin QR-Codes und der lange Code - Charaktere bleiben dann aber nur lokal.

Bibliothek

Hier findest du generische NSCs aus dem Regelwerk (Ganger, Soldaten, Geister, Critter etc.). Du kannst eigene Einträge hinzufügen oder bestehende anpassen.

Zustandsmonitore

Körperlicher Schaden (K) = rote Kreise
Geistiger Schaden (G) = gelbe Kreise
Panzerung = cyan Kreise (Schaden geht zuerst auf die Panzerung)

Jede volle Reihe (3 Kreise) ergibt einen Würfelmalus: Reihe 1 → −1, Reihe 2 → −2, usw. Die Malusse aus beiden Monitoren werden addiert. Voller Monitor = Außer Gefecht.

Daten-Speicherung

Alle Daten werden lokal im Browser gespeichert (LocalStorage). Wenn du den Browser-Cache leerst, sind die Daten weg. Nutze den Export-Code um Backups anzulegen.
